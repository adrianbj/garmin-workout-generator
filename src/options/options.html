Parser backend

By default the extension parses with Chrome's on-device Gemini Nano (free, no key, ~30s/parse, sometimes flubs notation). Paste a Google AI Studio API key to use Gemini 2.5 Flash instead (free tier covers ~1500 parses/day, ~1s/parse, far more reliable). Get a key at aistudio.google.com/apikey.

Pace zones

Used by the parser to resolve names like "aerobic threshold" to actual paces. Pace is m:ss per km.
